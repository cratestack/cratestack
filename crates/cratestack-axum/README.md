# cratestack-axum

Axum integration for generated routes and HTTP transport.

## Overview

`cratestack-axum` provides HTTP transport utilities, content negotiation, and Axum router generation for CrateStack services.

## Installation

```toml
[dependencies]
cratestack-axum = "0.2"
axum = "0.7"
```

## Usage

### Generated Router

Generated by `include_schema!`:

```rust
use cratestack::include_schema;
use cratestack_codec_cbor::CborCodec;

include_schema!("schema.cstack");

let pool = sqlx::PgPool::connect(&database_url).await?;
let cool = cratestack_schema::CrateStack::builder(pool).build();

let router = cratestack_schema::axum::model_router(
    cool,
    CborCodec,
    AppAuthProvider, // implements AuthProvider
);

let app = axum::Router::new().nest("/api", router);
```

### HTTP Transport

Implement the transport trait for content negotiation:

```rust
use cratestack_axum::HttpTransport;
use cratestack_core::CoolCodec;

fn handle_request<C: CoolCodec + HttpTransport>(codec: C, body: &[u8]) {
    // Decode based on Content-Type
    let input: CreateUserInput = codec.decode_request(content_type, body)?;
    
    // Process...
    
    // Encode response based on Accept
    let response = codec.encode_response(accept_type, StatusCode::OK, &result)?;
}
```

### Content Types

| Content Type | Codec |
|--------------|-------|
| `application/cbor` | CBOR single item |
| `application/json` | JSON single item |
| `application/cbor-seq` | CBOR sequence (streaming) |

### Content Negotiation

```rust
use cratestack_axum::media_type_matches;

if media_type_matches(content_type, "application/cbor") {
    // Handle CBOR
} else if media_type_matches(content_type, "application/json") {
    // Handle JSON
}
```

## Middleware

### Rate Limiting

Per-principal rate limiting:

```rust
use cratestack_axum::ratelimit::{RateLimitLayer, InMemoryRateLimitStore};

let store = InMemoryRateLimitStore::new();
let app = axum::Router::new()
    .route("/api", router)
    .layer(RateLimitLayer::new(store, 100, Duration::from_secs(60)));
```

### Idempotency

Duplicate execution protection:

```rust
use cratestack_axum::idempotency::{IdempotencyLayer, SqlxIdempotencyStore};

let store = SqlxIdempotencyStore::new(pool);
let app = axum::Router::new()
    .route("/api", router)
    .layer(IdempotencyLayer::new(store));
```

## Query Parsing

Parse filter expressions from URL query parameters:

```rust
use cratestack_axum::QueryExpr;

// ?filters[status]=active&filters[authorId]=123
let expr = QueryExpr::from_query_params(query_params);
```

## Transport Architecture

See [Transport Architecture](https://cratestack.dev/architecture/transport-architecture) for the full design:

1. **Codec**: CBOR/JSON serialization
2. **Framing**: Single item vs. sequence
3. **Envelope**: Optional signing (COSE planned)
