# Security Policy

CrateStack is pre-1.0. Treat all generated HTTP, policy, and client-runtime behavior as subject to breaking fixes when a security issue is found.

To report a vulnerability, email Stephane Segning at selastlambou@gmail.com with:

* affected version or commit
* reproduction steps
* expected and observed impact
* any suggested mitigation

Please avoid public disclosure until a fix or advisory is available.
